## Описание проекта

Данный проект представляет собой **ИИ-ассистента**, разработанного специально для компаний, занимающихся продажей портативных гаражей. Ассистент создан с целью автоматизации общения с клиентами, повышения качества обслуживания и оптимизации рабочих процессов.

---

## Функционал ИИ-ассистента

### 1. **Осмысленное ведение диалога с помощью нейросети**
   - ИИ-ассистент способен поддерживать естественную беседу с клиентами, отвечая на вопросы различной сложности.
   - Диалог строится на основе предварительно подготовленной базы знаний и адаптируется к контексту общения.

### 2. **Ответы на вопросы по базе знаний**
   - Ассистент предоставляет точные и релевантные ответы на часто задаваемые вопросы о продукции, условиях покупки, гарантиях и послепродажном обслуживании.
   - База знаний включает информацию о материалах, размерах, комплектации, ценах и других характеристиках портативных гаражей.

### 3. **Запись клиента на встречу**
   - Клиент может записаться на консультацию или осмотр образцов гаражей через ассистента.
   - В процессе записи собираются необходимые данные (дата, время, контактная информация).

### 4. **Добавление встречи в Google Календарь**
   - После успешной записи ассистент автоматически создает событие в Google Календаре, уведомляя как клиента, так и сотрудников компании о предстоящей встрече.

---

## Используемые сервисы

### 1. **Qwen**
   - **Назначение**: Создание системного промпта и базы знаний.
   - **Особенности**:
     - Qwen был использован для разработки структурированной базы знаний, содержащей информацию о продуктах, услугах и бизнес-процессах компании.
     - Системный промпт настроен таким образом, чтобы обеспечить корректное понимание запросов клиентов и предоставление качественных ответов.

### 2. **Savvy (https://suvvy.ai/)**
   - **Назначение**: Платформа для создания и управления ИИ-ассистентом.
   - **Особенности**:
     - Savvy позволяет интегрировать нейросетевые модели в готовый интерфейс для общения с клиентами.
     - Платформа обеспечивает удобную настройку функционала (диалоги, записи, интеграции с календарем) без необходимости глубоких технических знаний.

---

## Преимущества использования ИИ-ассистента

- **Экономия времени**: Автоматизация процессов общения и записи на встречи освобождает сотрудников компании для выполнения более сложных задач.
- **Повышение конверсии**: Клиенты получают быстрые и точные ответы на свои вопросы, что увеличивает вероятность совершения покупки.
- **Улучшение опыта взаимодействия**: Естественный диалог и оперативное решение вопросов делают общение с компанией более комфортным.
- **Широкие возможности масштабирования**: ИИ-ассистент может быть адаптирован для работы с большим количеством клиентов без потери качества обслуживания.

## Пример использования

### Сценарий 1: Ответы на вопросы
Клиент: _"Какой материал используется для изготовления портативных гаражей?"_  
Ассистент: _"Наши гаражи производятся из высокопрочной ткани с 5-слойной конструкцией, которая защищает автомобиль от холода, влаги и ветра. Этот материал успешно прошел тестирование в экстремальных условиях Якутии."_  

---

### Сценарий 2: Запись на встречу
Клиент: _"Можно записаться на осмотр гаража?"_  
Ассистент: _"Конечно! Когда вам будет удобно встретиться? Например, завтра в 15:00."_  
Клиент: _"Подходит."_  
Ассистент: _"Отлично! Я записал вас на завтра в 15:00. Подтверждаю встречу и добавляю её в ваш календарь."_

## Технические детали

- **Интеграция с Google Календарем**: Для автоматического создания событий используется API Google Calendar.
- **База знаний**: Обновляется регулярно сотрудниками компании для обеспечения актуальности информации.
- **Языковая поддержка**: Ассистент работает на русском языке, но может быть адаптирован для других языков при необходимости.
